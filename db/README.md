# scuderia-backend

Node.js + Express backend con SQLite per BRC_Home.

## Avvio su hosting (Render)
- Start command: `node server.js`
- Impostare le env vars su Render (Settings -> Environment):
  - `JWT_SECRET` (stringa segreta)
  - `ADMIN_EMAIL` (es. info@battagliaracingcar.com)
  - `ADMIN_PASSWORD` (es. Prova1234!)
